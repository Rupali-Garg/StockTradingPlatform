# Stock Trading & Portfolio Management Platform

A full-stack stock trading platform inspired by modern brokerage applications. The platform allows users to manage portfolios, track transactions, create watchlists, and monitor stock performance through an interactive dashboard.

## Features

* User Authentication
* Portfolio Management
* Stock Watchlists
* Transaction Tracking
* Dashboard Analytics
* Responsive User Interface
* REST API Integration

## Tech Stack

### Frontend

* React.js
* JavaScript
* CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Authentication

* JWT
